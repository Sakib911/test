### For New Developers

1. **Read the [Design System](./DESIGN_SYSTEM.md)** to understand the component library
2. **Review the [Cursor Rules](../.cursorrules)** for coding standards
3. **Check existing components** before creating new ones
4. **Follow the [Module Documentation Template](./MODULE_DOCUMENTATION_TEMPLATE.md)** for new features

### For Module Development

1. **Use the [Module Documentation Template](./MODULE_DOCUMENTATION_TEMPLATE.md)** as a starting point
2. **Follow SOLID principles** and design patterns
3. **Use existing UI components** from the design system
4. **Implement proper TypeScript types** and error handling
5. **Add comprehensive documentation** for your module
